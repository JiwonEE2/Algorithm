# C++ `auto` 키워드 요약 정리

---

## 📁 헤더

- `auto` 자체는 헤더 불필요  
- 컨테이너와 함께 사용 시 해당 헤더 포함

---

## 🛠️ 기본 사용법

```cpp
auto x = 10;               // int
auto y = 3.14;             // double
for (auto it = v.begin(); it != v.end(); ++it) { ... }
```

---

## ⚙️ 특징

- 초기화 값을 기반으로 타입 자동 추론  
- 반복자, 람다, 함수 반환값 등에 유용  
- `auto&`, `const auto&`로 참조 사용 가능  
- 반드시 초기화 필요

---

## 🧾 예시

```cpp
vector<int> v = {1, 2, 3};
for (auto x : v) cout << x << ' ';
```

---

## 💡 팁

- 긴 타입을 간결하게 표현  
- 값 복사보다 `auto&` 권장 (성능 + 수정 가능)

---

## 🧷 요약

- `auto`는 타입을 자동 추론  
- 반복자 등 복잡한 타입 간결하게 처리  
- 반드시 초기화 필요, 추론된 타입은 고정됨
