# 🔢 C++ 숫자형 자료형 정리

| 자료형         | 크기 (byte) | 범위 (약)                                      | 소수점 | signed/unsigned | 비고                              |
| -------------- | ----------- | ---------------------------------------------- | ------ | --------------- | --------------------------------- |
| `int`          | 4           | -2,147,483,648 \~ 2,147,483,647                | ❌     | 둘 다 가능      | 가장 일반적인 정수형              |
| `long`         | 4 or 8      | 시스템에 따라 다름 (보통 `int`와 같거나 더 큼) | ❌     | 둘 다 가능      | 64비트 환경에서는 보통 8바이트    |
| `long long`    | 8           | -9경 \~ 9경 (`±9.2×10¹⁸`)                      | ❌     | 둘 다 가능      | 매우 큰 정수 필요 시 사용         |
| `float`        | 4           | ±3.4×10³⁸, 소수점 6자리 정확도                 | ✅     | signed만        | 빠르지만 정밀도 낮음              |
| `double`       | 8           | ±1.7×10³⁰⁸, 소수점 15자리 정확도               | ✅     | signed만        | 더 정밀한 실수 연산               |
| `long double`  | 8\~16       | 구현마다 다름                                  | ✅     | signed만        | 매우 높은 정밀도 필요할 때 (드뭄) |
| `unsigned int` | 4           | 0 \~ 4,294,967,295                             | ❌     | unsigned        | 음수 없고 양수만 필요할 때 사용   |

> ❗ **크기(byte)나 범위는 시스템 및 컴파일러 구현에 따라 다를 수 있음**
> (C++ 표준은 `sizeof(int) ≥ sizeof(short)` 정도의 상대적 보장만 합니다)

---

## 📍 정수형 상세

- `short`: 2바이트 정수. 작은 범위 필요 시
- `int`: 대부분의 상황에서 정수는 이걸 사용
- `long`, `long long`: 큰 정수 필요 시
- `unsigned`: 범위를 두 배로 늘리는 대신 음수 제거

```cpp
unsigned int u = 4000000000; // OK
int x = -100;                // OK
```

---

## 📍 실수형 상세

| 타입          | 메모리  | 정밀도 (유효 자리수) | 표기 방식         |
| ------------- | ------- | -------------------- | ----------------- |
| `float`       | 4바이트 | 약 6자리             | 소수 또는 지수(e) |
| `double`      | 8바이트 | 약 15자리            | 보통 default      |
| `long double` | 12\~16  | 약 18자리 이상       | 정밀 계산용       |

```cpp
float a = 3.14159f;       // f 붙여야 float
double b = 3.1415926535;  // double
```

---

## 📎 표기법

```cpp
int i = 100;
float f = 3.14f;
double d = 2.718;
long long big = 10000000000LL;
unsigned int ui = 42U;
```

- `f` 또는 `F`: float 리터럴
- `U`: unsigned
- `L`: long
- `LL`: long long

---

## ✅ 요약

| 상황                         | 자료형 추천    |
| ---------------------------- | -------------- |
| 일반적인 정수                | `int`          |
| 큰 정수 필요                 | `long long`    |
| 음수 없이 큰 값 필요         | `unsigned int` |
| 빠르지만 덜 정확한 실수      | `float`        |
| 정밀한 실수 계산 필요        | `double`       |
| 매우 높은 정밀도 필요 (희귀) | `long double`  |

---

## 🧪 참고: `sizeof()`로 확인하기

```cpp
#include <iostream>
int main() {
    std::cout << sizeof(int) << "\n";
    std::cout << sizeof(long) << "\n";
    std::cout << sizeof(long long) << "\n";
    std::cout << sizeof(float) << "\n";
    std::cout << sizeof(double) << "\n";
}
```
